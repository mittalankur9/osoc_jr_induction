--version
Prints the Git suite version that the git program came from.

--info-path
Print the path where the Info files documenting this version of Git are installed and exit.

git init
This command turns a directory into an empty Git repository

git add
Adds files in the to the staging area for Git. Before a file is available to commit to a repository, the file needs to be added to the Git index (staging area).

git commit
Record the changes made to the files to a local repository. 

git status
This command returns the current state of the repository.
git status will return the current working branch.

git pull
To get the latest version of a repository run git pull. This pulls the changes from the remote repository to the local computer.

git-restore
Restore working tree files

git-mv
Move or rename a file, a directory, or a symlink
